# clinical-code-mapper

![Python](https://img.shields.io/badge/python-3.9+-blue?logo=python)
![License](https://img.shields.io/badge/License-MIT-green)

Python library for mapping between SNOMED-CT, ICD-10, CPT, and LOINC clinical coding systems with fuzzy search and hierarchical navigation

## Features

- Python library for mapping between SNOMED-CT, ICD-10, CPT, and LOINC clinical coding systems with fuzzy search and hierarchical navigation

## Installation

```bash
git clone https://github.com/DinhLucent/clinical-code-mapper.git
cd clinical-code-mapper
pip install -r requirements.txt
```
